`dnsmasq` service je i DHCP client i DNS server.

U `/etc/resolv.conf` mozemo vidjeti IP DNS servera.

`dig five.foo.lan`

`local=/foo.lan/`

```
noresolv
server=8.8.8.8
```

`/etc/hosts`, potreban restart.
